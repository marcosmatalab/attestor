Documentation assets for the root README.

## `dashboard.png`

A live capture of the dashboard's end-to-end demo page (`/demo`), referenced from the root
README. It shows real engine output — the classification checksum, the C2PA report with the
dev signer reported **untrusted**, and the ledger **verified** offline. It is a genuine
screenshot of the running app, not a mockup.

To re-capture it: start the backend and frontend (see
[Run it locally](../README.md#run-it-locally)), open http://localhost:3000/demo, run the demo,
and save the page as `docs/dashboard.png`.
